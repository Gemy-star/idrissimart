# 🛍️ إدريسي مارت (Idrissi Mart)

**إدريسي مارت** هو منصة تجارة إلكترونية حديثة مبنية بإطار **Django**، تهدف إلى تمكين المتاجر والبائعين من عرض منتجاتهم بسهولة وبأسلوب عصري يدعم اللغة العربية والإنجليزية، مع تصميم متجاوب وأداء قوي وتحسينات أمنية متقدمة.

---

## 🌟 المميزات الرئيسية

### 🎨 **واجهة المستخدم**
- تصميم حديث ومتجاوب باستخدام **Bootstrap 5 + GSAP + Swiper.js**
- دعم **الوضع الليلي/النهاري** مع حفظ التفضيل تلقائيًا
- **محول لغات** (عربي / إنجليزي) مع اتجاه ديناميكي RTL/LTR
- حركات انتقالية ناعمة وتأثيرات Parallax
- أزرار متحركة بتدرجات ألوان جميلة

### ⚙️ **الوظائف الأساسية**
- إدارة البائعين والمنتجات بسهولة
- صفحات ديناميكية مع تصنيفات وفلاتر
- دعم كامل لملفات الوسائط والصور
- إعدادات بيئية آمنة باستخدام `.env`
- نظام إعدادات منفصل (محلي / إنتاج / Docker)

### 🔒 **الأمان والجودة**
- فحص الأمان عبر **Bandit**
- تحسين جودة الكود باستخدام **Ruff**
- فحص القوالب عبر **DjLint**
- جميع المفاتيح والبيانات السرية تُدار من `.env`
- جميع الفحوصات تمر بنجاح ✅

### 🧑‍💻 **تجربة المطورين**
- أوامر سريعة عبر **Makefile**
- إعداد CI/CD عبر GitHub Actions
- فحوصات Pre-commit تلقائية
- توثيق كامل وخطة تحسين جاهزة

## ⚙️ التثبيت والإعداد

### **1. تثبيت المتطلبات**
```bash
pip install -r requirements.txt
```

أو إذا كنت تستخدم Poetry:
```bash
poetry install
```

---

### **2. إعداد ملف البيئة**
انسخ القالب:
```bash
cp .env.example .env
```

أنشئ مفتاح أمان:
```bash
python -c "from django.core.management.utils import get_random_secret_key; print(get_random_secret_key())"
```

ثم أضف المفتاح داخل `.env`:
```
DJANGO_SECRET_KEY=ضع_المفتاح_هنا
DB_PASSWORD=كلمة_مرور_قاعدة_البيانات
```

---

### **3. تهيئة قاعدة البيانات**
```bash
python manage.py migrate
```

---

### **4. تشغيل الخادم المحلي**
```bash
python manage.py runserver
```

ثم افتح المتصفح على:
```
http://127.0.0.1:8000/
```

## 🚀 أوامر Makefile المفيدة

| الأمر | الوصف |
|-------|--------|
| `make help` | عرض جميع الأوامر |
| `make install` | تثبيت جميع التبعيات |
| `make setup` | إعداد المشروع بالكامل |
| `make lint` | تشغيل جميع أدوات الفحص |
| `make test` | تشغيل الاختبارات |
| `make run` | تشغيل الخادم المحلي |
| `make clean` | تنظيف الملفات المؤقتة |

## 🧪 فحوصات الجودة

### **تشغيل أدوات الفحص يدويًا**
```bash
ruff check .
ruff format .
bandit -r . -c pyproject.toml
djlint templates/ --check
```

### **تشغيل الاختبارات**
```bash
python manage.py test
```

### **التحقق العام**
```bash
python manage.py check
```

## 🌍 اللغات والدعم الدولي

- اللغة الافتراضية: **العربية (RTL)**
- مدعومة أيضًا: **الإنجليزية (LTR)**
- يتم حفظ تفضيل اللغة محليًا
- يدعم Django i18n مع أوامر:
  ```bash
  python manage.py makemessages -l ar
  python manage.py makemessages -l en
  python manage.py compilemessages
  ```
