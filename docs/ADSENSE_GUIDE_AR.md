# دليل البنرات المدفوعة + Google AdSense — الشرح الكامل للنظام

---

## أولاً: فهم النموذج التجاري كاملاً

### مصادر الدخل من مساحات الإعلانات

الموقع يملك **مساحات إعلانية** في مواضع محددة بكل صفحة. هذه المساحات تُدرّ دخلاً بطريقتين:

```
┌─────────────────────────────────────────────────────────────┐
│                   مساحة إعلانية (موضع)                      │
├─────────────────────────┬───────────────────────────────────┤
│   إعلان مدفوع (Paid)    │      Google AdSense (احتياطي)     │
│                         │                                   │
│  صاحب نشاط يدفع مبلغاً  │  Google يعرض إعلاناتها ويدفع    │
│  ثابتاً شهرياً للموقع   │  عمولة لكل ألف مشاهدة (CPM)     │
│                         │                                   │
│  مثال: 200 جنيه/شهر     │  مثال: 3-10 دولار/ألف مشاهدة   │
└─────────────────────────┴───────────────────────────────────┘
```

**قاعدة النظام**: الإعلان المدفوع دائماً له الأولوية. AdSense يعمل **فقط** حين لا يوجد إعلان مدفوع نشط في ذلك الموضع.

## ثالثاً: رحلة الإعلان المدفوع — من البيع للعرض

### كيف يشتري العميل إعلاناً مدفوعاً؟

```
1. العميل يختار نوع البنر (بانر / جانبي / مميز)
      ↓
2. يختار نوع الموضع (رئيسية / قسم / قسم فرعي)
      ↓
3. النظام يعرض السعر من جدول BannerPricing
   (مثال: بانر + رئيسية = 200 جنيه/شهر)
      ↓
4. العميل يرفع صورة الإعلان ويحدد:
   - الرابط عند الضغط
   - تاريخ البداية والنهاية
   - الدولة المستهدفة (مصر / السعودية / ...)
      ↓
5. الإعلان ينتظر موافقة الأدمن (حالة: Pending)
      ↓
6. الأدمن يوافق → الإعلان يصبح Active
      ↓
7. الإعلان يظهر على الموقع خلال تواريخ الصلاحية
```

### متى يتوقف الإعلان؟

- انتهت تواريخ الصلاحية → الحالة تصبح `Expired`
- أوقفه الأدمن يدوياً → الحالة تصبح `Paused`
- في كلتا الحالتين → **AdSense يأخذ Slot مكانه تلقائياً**

## خامساً: ربط Google AdSense بموضع معين

### المواضع المتاحة لـ AdSense

| Slot Key | يقابله |
|---|---|
| `home_banner` | بانر الصفحة الرئيسية |
| `home_sidebar` | جانبي الصفحة الرئيسية |
| `home_featured` | مميز الصفحة الرئيسية |
| `category_banner` | بانر صفحات الأقسام |
| `category_sidebar` | جانبي صفحات الأقسام |
| `category_featured` | مميز صفحات الأقسام |

### خطوات التفعيل

**الخطوة 1 — أنشئ وحدة إعلانية في Google AdSense**

1. ادخل [Google AdSense](https://adsense.google.com)
2. **الإعلانات ← حسب الوحدة الإعلانية ← + جديد**
3. اختر النوع المناسب للموضع (أفقي للبانر، مستطيل للجانبي)
4. احتفظ بـ:
   - `ca-pub-XXXXXXXXXXXXXXXX` ← Publisher ID
   - `XXXXXXXXXX` ← Ad Slot ID (الرقم فقط)

**الخطوة 2 — أضف الإعداد في لوحة التحكم**

اذهب إلى: `/super-admin/main/adsenseslot/add/`

| الحقل | القيمة |
|---|---|
| **Slot Key** | الموضع المطلوب (مثل `home_banner`) |
| **Country** | الدولة (مصر، السعودية، ...) |
| **Ad Client** | `ca-pub-XXXXXXXXXXXXXXXX` |
| **Ad Slot** | الرقم فقط مثل `9876543210` |
| **Ad Format** | `auto` في الغالب (أو `horizontal` للبانر) |
| **Is Responsive** | ✓ |
| **Is Active** | ✓ |

**الخطوة 3 — تحقق**

- افتح الصفحة وتأكد أنه لا يوجد إعلان مدفوع نشط في ذلك الموضع
- يجب أن يظهر إطار AdSense مكانه فوراً
- قد يستغرق Google بضع ساعات لبدء عرض إعلانات حقيقية على موقع جديد

## سادساً: سيناريوهات عملية

### سيناريو 1 — الموضع فارغ تماماً

```
لا يوجد إعلان مدفوع + لا يوجد إعداد AdSense
  → المساحة لا تظهر أصلاً (مخفية)
```

### سيناريو 2 — AdSense فقط

```
لا يوجد إعلان مدفوع + يوجد إعداد AdSense نشط
  → Google AdSense يظهر في المساحة
  → الموقع يكسب عمولة CPM من Google
```

### سيناريو 3 — إعلان مدفوع نشط

```
يوجد إعلان مدفوع نشط + يوجد إعداد AdSense
  → الإعلان المدفوع يظهر (له الأولوية)
  → AdSense مخفي حتى ينتهي الإعلان
```

### سيناريو 4 — انتهى الإعلان المدفوع

```
انتهت صلاحية الإعلان المدفوع + يوجد إعداد AdSense
  → الإعلان المدفوع يختفي تلقائياً
  → AdSense يظهر مكانه فوراً بدون تدخل
```

## ثامناً: ملاحظات مهمة

- **كل دولة إعداد منفصل** — يمكن إعداد AdSense لمصر فقط وترك باقي الدول بدون إعلانات
- **لا ازدواجية** — لا يمكن إضافة نفس `Slot Key` لنفس الدولة مرتين
- **إيقاف AdSense** — ببساطة قم بإلغاء تفعيل `Is Active` من لوحة التحكم
- **معاينة الكود** — في صفحة تعديل أي Slot يوجد حقل "معاينة كود AdSense" يعرض الكود الجاهز

## فكرة عمل النظام

النظام يعمل بمبدأ **الأولوية للإعلان المدفوع**:

```
هل يوجد إعلان مدفوع نشط لهذا الموضع؟
    ✅ نعم  →  اعرض الإعلان المدفوع (يدفع العميل مباشرة للموقع)
    ❌ لا   →  اعرض إعلان Google AdSense (يدفع Google للموقع)
```

أي أنك **لا تخسر مساحة الإعلان أبداً** — إما دخل مباشر من عميل، أو دخل تلقائي من Google.

## خطوات تفعيل AdSense على موضع معين

### الخطوة 1 — احصل على بيانات AdSense من لوحة Google

1. ادخل على [Google AdSense](https://adsense.google.com)
2. اذهب إلى **الإعلانات ← حسب الوحدة الإعلانية**
3. اضغط **+ جديد** وأنشئ وحدة إعلانية بالحجم المناسب للموضع
4. احتفظ بـ:
   - **Publisher ID** — يبدأ بـ `ca-pub-` مثل `ca-pub-1234567890123456`
   - **Slot ID** — رقم مثل `9876543210`

### الخطوة 2 — أضف الإعداد في لوحة التحكم

1. ادخل على: `/super-admin/main/adsenseslot/add/`
2. املأ الحقول:

| الحقل | القيمة |
|---|---|
| **Slot Key (موضع الإعلان)** | اختر الموضع المطلوب من القائمة |
| **Country (الدولة)** | اختر الدولة (مصر، السعودية، إلخ) |
| **Ad Client** | `ca-pub-XXXXXXXXXXXXXXXX` |
| **Ad Slot ID** | رقم الـ slot من AdSense |
| **Ad Format** | `auto` (تلقائي — الأفضل في الغالب) |
| **Is Responsive** | ✓ مفعّل (لضمان التوافق مع الجوال) |
| **Is Active** | ✓ مفعّل |

3. اضغط **حفظ**

### الخطوة 3 — تحقق من ظهور الإعلان

- افتح الصفحة المقابلة وتأكد أنه **لا يوجد إعلان مدفوع نشط** لذلك الموضع
- يجب أن يظهر إعلان Google مكانه تلقائياً
- قد يستغرق Google بضع ساعات لبدء عرض الإعلانات على موقع جديد

## ملاحظات مهمة

- **كل دولة إعداد منفصل** — يمكنك استخدام slot مختلف لمصر وآخر للسعودية

```
مثال: الصفحة الرئيسية — 10,000 زيارة شهرياً

إذا كان هناك إعلان مدفوع نشط طوال الشهر:
```
